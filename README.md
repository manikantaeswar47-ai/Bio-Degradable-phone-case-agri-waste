# Bio-Degradable Phone Case (Agri Waste)

## Overview

* Eco-friendly product-based full-stack project
* Uses agricultural waste to create biodegradable phone cases
* Reduces plastic pollution and promotes sustainability
* Encourages green and eco-conscious technology solutions

## Features

* Product information display system
* User registration and login
* Order and inquiry management
* Eco-friendly material details section
* Contact and feedback system

## Process

* User visits the platform
* Registers or logs in
* Views biodegradable product details
* Places order or sends inquiry
* Receives confirmation or response

## Advantages

* Reduces environmental pollution
* Promotes sustainable development
* Cost-effective eco-friendly solution
* Supports reuse of agricultural waste

## Limitations

* Limited durability compared to plastic cases
* Requires awareness among users
* Production scalability challenges

## Future Scope

* Large-scale eco-product manufacturing
* Mobile app integration
* AI-based product customization
* Expansion to other biodegradable products
